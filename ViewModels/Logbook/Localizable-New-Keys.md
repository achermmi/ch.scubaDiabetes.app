# Nuove chiavi di localizzazione da aggiungere

Queste chiavi devono essere aggiunte ai file di localizzazione dell'app (tipicamente `Localizable.strings` o `Localizable.xcstrings`).

## Profilo Salute

```
// Unità glicemia
"profile.health.glucose_unit" = "Unità glicemia";

// Privacy
"profile.health.privacy" = "Privacy e Ricerca";
"profile.health.share_for_research" = "Condividi dati per la ricerca";
"profile.health.share_for_research_desc" = "I tuoi dati potranno essere utilizzati in forma anonima per studi scientifici";
"profile.health.privacy_footer" = "Questa impostazione definisce il comportamento predefinito per tutte le tue immersioni. Potrai comunque modificare la privacy di ogni singola immersione.";
```

## Form Immersione

```
// Privacy immersione
"dive.form.privacy" = "Privacy";
"dive.form.share_for_research" = "Condividi questa immersione per la ricerca";
"dive.form.share_for_research_desc" = "Permetti l'uso anonimo di questi dati per studi scientifici";
```
